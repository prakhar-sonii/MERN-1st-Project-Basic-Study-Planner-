# Smart Study Planner

A web-based application designed to help students organize their study schedules, manage subjects, and track tasks efficiently. This project utilizes local storage to persist data, ensuring your study plan remains saved between sessions.

## Features

- **Dashboard**: A central hub for an overview of your study activities.
- **Subject Management**: 
  - Add subjects with specific priority levels (High, Medium, Low).
  - View a list of current subjects.
  - Delete subjects when no longer needed.
- **Task Manager**: 
  - Create tasks with specific deadlines.
  - (In Progress) View and manage task completion.
- **Schedule Planner**: A section dedicated to daily and weekly planning.
- **Settings**:
  - **Dark Mode**: Toggle between light and dark themes for better visibility.
  - **Reset Data**: Clear all stored data to start fresh.
- **Data Persistence**: All subjects and tasks are saved to the browser's LocalStorage.


## Technologies Used

- **HTML5**: Semantic structure of the application.
- **CSS3**: Styling and layout (including Dark Mode support).
- **JavaScript (ES6)**: Logic for handling user input, navigation, and data storage.
